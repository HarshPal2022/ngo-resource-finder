# NGOConnect AI

An AI-powered NGO Discovery and Recommendation Platform built using:

- Python
- Flask
- MySQL
- BeautifulSoup
- Pandas
- Scikit-learn
- Tailwind CSS

## Features

- NGO Search
- AI Recommendations
- Web Scraping Pipeline
- MySQL Storage
- REST APIs
- Responsive Frontend
- Render Deployment
